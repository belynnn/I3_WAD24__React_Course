## Notes
### Créer une application React + Vite
1. Créer le projet :
    ```
    npm create vite@latest ReactApp
    ```
2. Aller dans le dossier du projet :
    ```
    cd ReactApp
    ```
3. Installer node_modules :
    ```
    npm i
    ```
4. Lancer l'application en mode Développement :
    ```
    npm run dev
    ```
